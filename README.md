# Hand-Gesture-Recognition-System
A real-time Hand Gesture Recognition System built with Python, OpenCV, and MediaPipe. It detects and tracks hand landmarks, recognizes finger positions, and enables gesture-based interaction through a webcam. Designed to demonstrate computer vision, image processing, and AI concepts in an interactive way.
